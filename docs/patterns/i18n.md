# Pattern: i18n formatting

`ui-detox/i18n` provides locale-aware number / date / unit formatting on top of
`Intl` — no bundle bloat, no message catalogs (formatting only for MVP).

```ts
import { setLocale, registerUnit, fmt } from 'ui-detox/i18n';

setLocale('uk-UA');

registerUnit('g',    { symbol: 'г',    base: 1,    dimension: 'mass' });
registerUnit('mg',   { symbol: 'мг',   base: 1e-3, dimension: 'mass' });
registerUnit('mcg',  { symbol: 'мкг',  base: 1e-6, dimension: 'mass' });
registerUnit('kcal', { symbol: 'ккал', base: 1,    dimension: 'energy' });
```

Units sharing a `dimension` are inter-convertible and auto-scale together.

## Formatters

```ts
fmt.number(1234.5)                    // "1 234,5"
fmt.number(1234.5, { decimals: 0 })   // "1 235"
fmt.percent(0.82)                     // "82 %"
fmt.percent(0.821, { decimals: 1 })   // "82,1 %"
fmt.delta(12.3)                       // "+12,3"
fmt.delta(-4.5, { decimals: 1 })      // "-4,5"
fmt.qty(1234.5, 'mg')                 // "1 234,5 мг"
fmt.qty(1234.5, 'mg', { to: 'g' })    // "1,2 г"
fmt.qty(1234.5, 'mg', { auto: true }) // "1,2 г"  (best unit by magnitude)
fmt.qty(0.3, 'mg', { auto: true })    // "300 мкг"
fmt.date(iso)                         // "26 черв. 2026"
fmt.dateTime(iso)                     // "26 черв. 2026, 14:32"
fmt.relative(iso)                     // "2 години тому"
```

Auto-scale picks the unit where the value lands in `[0.5, 999]`.

## In templates

Call directly in an interpolation:

```html
<td>${fmt.qty(node.massNetoG, 'g')}</td>
<td>${fmt.qty(nutrient.amount, 'mg', { auto: true })}</td>
```

Or register filters and pipe:

```ts
import { registerI18nFilters } from 'ui-detox/i18n';
registerI18nFilters(); // number, percent, delta, qty, date, dateTime, relative
```

```html
<td>${node.massNetoG | qty:'g'}</td>
```

## String catalog (stub)

Translation is out of MVP scope, but a `t()` stub exists so call sites don't
change when a catalog arrives:

```ts
import { t, setCatalog } from 'ui-detox/i18n';
t('recipe.emptyGraph');            // → 'recipe.emptyGraph' (key echoed)
setCatalog({ 'recipe.emptyGraph': 'Порожній граф' });
t('recipe.emptyGraph');            // → 'Порожній граф'
```
